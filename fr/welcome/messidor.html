Utiliser les images de MESSIDOR

Les images de fond d’œil annotées dans MAPLES-DR sont la propriété du consortium MESSIDOR et ne sont donc pas inclues dans le jeu de données. Cependant elles sont accessible librement à toutes les équipes de recherche qui en font la demande sur le site du consortium. La présente page fournit les instructions pour inclure les images de MESSIDOR avec les annotations de MAPLES-DR à l’aide de la bibliothèque python maples_dr.

Télécharger les images de fond d’œil de MESSIDOR

Les images de MESSIDOR peuvent-être téléchargées depuis le site du consortium. Vous devrez remplir un formulaire avec vos informations personnelles, accepter les termes de licence, et vérifier votre adresse email pour accéder à la page de téléchargement.

La page de téléchargement contient des liens vers de plusieurs fichiers. Les images de fond d’œil ont été divisées en 12 archives zip appelées Base__ images (zip). Assurez-vous de télécharger les 12 archives (Base11.zip, Base12.zip, …, Base34.zip) dans un dossier unique.

Configurer le chemin vers MESSIDOR

Une fois les images téléchargées, vous pouvez les inclure dans le dataset avec:

from maples_dr import configure
configure(messidor_path='/path/to/messidor/download/directory/')

Lors de l’appel de maples_dr.load_train_set() ou de maples_dr.load_test_set(), les 200 images de fond d’œil annotées dans MAPLES-DR seront extraites, découpées et redimensionnées pour correspondre à la résolution des cartes de segmentation de MAPLES-DR. Ces images de fond d’œil seront accessibles de la même manière que les autres biomarqueurs de MAPLES-DR, sous le nom "fundus".

from maples_dr import load_train_set
train_set = load_train_set()
fundus = train_set[0]["fundus"]

De façon analogue, lors d’un export de MAPLES-DR vers des dossiers locaux avec maples_dr.export_train_set() ou maples_dr.export_test_set(), un dossier appelé fundus sera créé à coté de ceux des autres biomarqueurs, contenant les images de fond d’œil à la résolution adéquate.

Remerciements

Le jeu de données d’images de fond d’œil est fourni par les partenaires du programme Messidor (voir https://www.adcis.net/fr/logiciels-tiers/messidor-fr/).

Nous rappelons que les utilisateurs de la base de données MESSIDOR (et donc les utilisateurs de MAPLES-DR) sont encouragés à citer l’article suivant:

Decencière et al.. Feedback on a publicly distributed database: the Messidor database.
Image Analysis & Stereology, v. 33, n. 3, p. 231-234, aug. 2014. ISSN 1854-5165.
Available at: http://www.ias-iss.org/ojs/IAS/article/view/1155 or http://dx.doi.org/10.5566/ias.1155.
